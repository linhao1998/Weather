### 新增功能

- 提供更加完整的天气信息，包括：
  - 新增体感温度，风向，风力和湿度实时天气信息；
  - 新增24小时预报功能；
  - 将5天预报修改为7天预报。

- 新增地点管理功能，可以对常用地点进行保存；
- 对深色主题进行适配。

### 新增技术栈

- 使用自定义View绘制实时天气信息View；
- 使用Room框架对存储的地点信息进行管理。
